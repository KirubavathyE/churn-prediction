# Customer Churn Prediction Using Machine Learning

## Project Overview

Customer churn refers to customers discontinuing a company's services. This project uses Machine Learning to predict whether a customer is likely to leave based on customer demographics, account information, and service usage patterns.

The goal is to help businesses identify at-risk customers and improve customer retention strategies.

## Features

* Data Cleaning and Preprocessing
* Handling Missing Values
* Feature Engineering
* Customer Churn Prediction using Random Forest Classifier
* Model Performance Evaluation
* Feature Importance Analysis
* Data Visualization using Matplotlib and Seaborn

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Dataset

Dataset: IBM Telco Customer Churn Dataset

The dataset contains customer information such as:

* Gender
* Senior Citizen Status
* Tenure
* Monthly Charges
* Total Charges
* Internet Service
* Contract Type
* Payment Method
* Churn Status

## Project Workflow

1. Load the dataset
2. Clean and preprocess data
3. Convert categorical variables into numerical format
4. Split data into training and testing sets
5. Train Random Forest Classifier
6. Predict customer churn
7. Evaluate model performance
8. Analyze feature importance

## Model Used

Random Forest Classifier

## Evaluation Metrics

* Accuracy Score
* Classification Report
* Confusion Matrix

## Key Insights

* Tenure is one of the strongest indicators of customer churn.
* Monthly charges significantly influence customer retention.
* Contract type impacts the likelihood of customer churn.
* Feature importance analysis helps identify critical churn factors.

## Results

The model successfully predicts customer churn and provides valuable insights that can help businesses improve customer retention strategies.
